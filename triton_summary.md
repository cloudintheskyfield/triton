# Triton 项目总结

Triton 是一种用于编写高效、自定义深度学习原语（primitives）的编程语言和编译器。它的核心目标是提供一个开源开发环境，让开发者能够以比 CUDA 更高的生产力编写高性能代码，同时提供比其他现有领域特定语言 (DSL) 更强的灵活性。

## 核心要点

*   **项目定位**：专门针对自定义深度学习计算操作设计的语言与编译器。
*   **主要优势**：兼顾高性能、高开发效率与高度的灵活性。
*   **安装方式**：
    *   可以通过 `pip install triton` 直接安装稳定版的预编译 Wheel 包。
    *   也支持克隆仓库从源码编译安装。
*   **编译后端**：Triton 依赖 LLVM 为 GPU 和 CPU 生成机器码。支持使用默认下载的预构建 LLVM，或者配置环境变量使用自定义编译的 LLVM 版本。
*   **开发与测试**：
    *   提供了 `make test` (需 GPU) 和 `make test-nogpu` (无需 GPU) 快捷指令来运行测试用例。
    *   提供了极其丰富的环境变量控制选项（如 `MLIR_ENABLE_DUMP`, `TRITON_INTERPRET` 等），方便开发者导出中间层代码(IR)、在解释器模式下设置 Python 断点调试 Kernel，或者深入检查 LLVM 的编译过程。
*   **平台兼容性**：目前主要支持 Linux 操作系统。
*   **硬件支持**：
    *   **NVIDIA GPUs**：支持计算能力 (Compute Capability) 8.0 及以上的硬件。
    *   **AMD GPUs**：支持 ROCm 6.2 及以上版本。
    *   **CPUs**：支持正在积极开发中。
*   **社区生态**：全面开源并积极欢迎开发者提交 Bug 修复或新功能 PR，提供了完整的开发容器（Dev Container）方案以统一开发环境。
